# Vankudothu Nithin - AI & Full-Stack Developer Portfolio

Welcome to the repository of my professional developer portfolio. This is a premium, highly interactive single-page application built using vanilla frontend technologies, featuring a sleek, futuristic dark-theme design with glassmorphic cards and glowing ambient lights.

🔗 **Live Deployment**: **[portfolio-three-iota-phrp0acbjm.vercel.app](https://portfolio-three-iota-phrp0acbjm.vercel.app)**

---

## 🌟 Key Features

* **Interactive Simulated RAG Terminal**: A custom command-line console where visitors can select quick prompts or type custom queries to retrieve details from my resume. It simulates a vector-index search and streams responses word-by-word with styled inline citation badges.
* **Typing Animation**: Interactive animated header in the hero section displaying dynamically changing developer roles.
* **Python Code Summary Card**: A stylized terminal window inside the "About Me" section showcasing developer characteristics written in structured Python OOP code.
* **Scroll-Triggered Reveals**: Uses high-performance `IntersectionObserver` to trigger elegant fade-in transitions as the visitor navigates the page.
* **Education & Certifications Timelines**: Beautiful timeline nodes illustrating my academic timeline alongside dedicated cards showcasing professional certifications.
* **Modern Contact Form**: Styled form utilizing input focus glow effects, validation, and simulated mail-routing feedback alerts.

---

## 🛠️ Technology Stack

* **Structure & Semantic Layout**: HTML5
* **Styling & Aesthetics**: Vanilla CSS3 (Custom Variables, Flexbox, Grid, Glassmorphic panels, Keyframe animations)
* **Interactivity & Logic**: JavaScript (ES6+, asynchronous timers, scroll observation API)
* **Icons & Fonts**: Google Fonts (Plus Jakarta Sans & JetBrains Mono), FontAwesome 6 icons
* **Deployment**: Hosted on **Vercel**

## 🚀 Running Locally

You can launch and review the portfolio locally using any basic HTTP web server.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nithin-0718/Portfolio.git
   cd Portfolio
   ```

2. **Start a local development server**:
   * **Using Python**:
     ```bash
     python -m http.server 8000
     ```
   * **Using Node (npx)**:
     ```bash
     npx serve
     ```

3. **View the website**:
   Open your browser and navigate to **`http://localhost:8000`** (or the port specified by your server).
